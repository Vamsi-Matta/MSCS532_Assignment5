# Quicksort Comparative Study

## Overview
This project explores two versions of the Quicksort algorithm:
- a fixed pivot implementation
- a randomized pivot implementation

The purpose of this assignment is to study how pivot selection affects the overall performance of Quicksort.

## Files Included
- `pivot_quicksort.py` - fixed pivot Quicksort implementation
- `random_pivot_sort.py` - randomized pivot Quicksort implementation
- `quicksort_experiment.py` - performance testing file
- `report.docx` - detailed report and analysis

## How to Run
Use Python to run each file:

```bash
python pivot_quicksort.py
python random_pivot_sort.py
python quicksort_experiment.py
